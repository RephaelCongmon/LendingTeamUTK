    About BrawlhallaStats.com

    What is this site?

    BrawlhallaStats.com is a standard statistics website for Brawlhalla.
    It has a minimalist design and is extremely easy to use.
    This website is still in early beta so there might be bugs wandering around. You can report them if you want in the official Discord server!

    I am consistently adding more features as time goes on.

    FAQ

    Where is this data coming from?
    - Most of the raw data are coming from the Official Brawlhalla API. All figures are calculated using the data given from that API.
    How can I get in contact with the developer?
    - You can contact me by emailing contact@brawlhallastats.com or by joining the Discord server.
    How often does the data get refreshed?
    - The speed of new data or updated data depends on the Brawlhalla API. It is usually within a couple minutes.
    How do I find my Brawlhalla ID?
    - You can find your Brawlhalla ID in the top right corner of the inventory in-game.
    Why do the Global Stats not seem accurate?
    - Because of the API Rate limit, I cannot search through every player and grab their total damage and update it every second. My solution is to just update global stats when a player has been searched. If you want to "pulse" a player's stats to the global statistics, just search them up and global stats will update.

    Future Updates

    More Global Statistics
    - Such as Total Damage Dealt "Globally" meaning the cumulative amount dealt from all players in the database. For this, I'm going to have to use my own server database and sync players up as they search (call it a "pulse") and update my own server database then.
