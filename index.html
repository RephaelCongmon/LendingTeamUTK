    What is this site?

    https://lendingteamutk.herokuapp.com is a website dashboard tool for the city of Knoxville that allows users to track and analyze public scooter transit statistics in an easy-to-navigate manner. Some useful features include:
    • Easy-To-Navigate website dashboard for all scooter transit data.
    • Easy access scooter transit heatmap.
    • A map of clusters displaying high scooter traffic and end points ideal for scooter parking
    • Works for all devices, mobile and desktop.
    How to Use this Site?
    Navigate the side menu to the scootLib data that you wish to see.
